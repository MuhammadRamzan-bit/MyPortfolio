Data Upload App (Android App)

Description:

A mobile application that allows users to upload and store data securely in the cloud using Firebase, ensuring accessibility and backup.


Key Features:

  • Simple form-based data entry
  • Firebase integration for real-time storage
  • Data retrieval and display functionality
  • Secure cloud-based storage

Tools & Technologies:

  1. Language: Kotlin
  2. Platform: Android Studio
  3. Database: Firebase
