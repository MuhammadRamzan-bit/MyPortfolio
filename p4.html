Famous Brand Shoes (Web App)

Description:

A responsive web-based inventory management system developed for a shoe brand to track stock levels, sales, and product details efficiently.


Key Features:

  • Product listing with categories and search
  • Add, update, and delete products easily
  • Stock management and low-stock alerts
  • Sales record and inventory reports

Tools & Technologies:

  1. Frontend: HTML, CSS, JavaScript
  2. Backend: JavaScript (Node.js or placeholder for backend)
  3. Database: MySQL (or dummy placeholder if not fully implemented)
