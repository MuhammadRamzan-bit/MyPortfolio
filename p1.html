Bus Management & Seat Reservation System

Description:

A desktop application to manage bus schedules, driver details, and seat reservations for passengers.


Key Features:

  • Add and manage bus and driver details
  • Seat booking and cancellation system
  • Passenger record management
  • Search and filter options for trips

Tools & Technologies:

  1. Language: Java (Java Swing for GUI)
  2. Platform: Desktop Application
  3. Database: MySQL
