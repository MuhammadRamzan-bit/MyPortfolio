BMI Calculator (Android App)

Description:

A simple yet efficient mobile application that calculates Body Mass Index (BMI) based on user input for height and weight, helping users track their fitness levels.


Key Features:

  • User-friendly interface for quick input
  • Instant BMI calculation and category display (Underweight, Normal, Overweight, etc.)
  • Real-time result updates without page reload
  • Lightweight and responsive app design

Tools & Technologies:

  1. Language: Java
  2. Platform: Android Studio
  3. Database: None (Local calculations)
